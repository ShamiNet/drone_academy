# 🚀 Quick Fix Reference

## The Error
```
API Error [LOGIN_FAIL]: Status: 400 - INVALID_LOGIN_CREDENTIALS
```

---

## What You Need to Do

### Option 1: Create a New Account ✅ **RECOMMENDED**

1. Open the app
2. Click **"Create New Account"**
3. Enter:
   - Email: `your.email@example.com`
   - Password: `SecurePassword123` (6+ characters)
   - Display Name: `Your Name`
   - Select Role: `Trainee` or `Trainer`
4. Click **Sign Up**
5. You'll be automatically logged in!

### Option 2: Fix Your Login Credentials

If you have an existing account:

1. **Check your email:** Is it spelled correctly?
2. **Check your password:** Is it the correct password?
3. **No spaces:** Remove any spaces before/after
4. **Try again**

## Still Having Issues?

### 1️⃣ Can't Remember Password?
→ Try creating a new account with a different email

### 2️⃣ Server Not Responding?
→ Check your internet connection
→ Try again in a few minutes
→ Server might be temporarily down

### 3️⃣ Device is Banned?
→ You'll see: `⛔ Device Banned`
→ Contact your administrator

### 4️⃣ Not Sure What's Wrong?
→ Read [LOGIN_DEBUGGING_GUIDE.md](LOGIN_DEBUGGING_GUIDE.md)
→ Follow the step-by-step troubleshooting
